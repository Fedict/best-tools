About BeST

The BeST-project contains the location of all current address points in Belgium, using the XML-export files provided by the 3 Regions.

The source code for this application is available as well.

Data

This service uses address data obtained from the 3 Regions in Belgium:

  • Brussels Capital Region (BRIC): UrbiS open license
  • Flemish Region (Digitaal Vlaanderen): Modellicentie gratis hergebruik
  • Walloon Region (SPW): Creative Commons Attribution
